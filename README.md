# 📈 TradeX – Zerodha-Inspired Stock Trading Platform

A full-stack stock trading platform inspired by **Zerodha**, built using **React.js**, **Node.js**, **Express**, and **MongoDB**. It replicates essential stock trading functionalities including **user authentication**, **stock buy/sell**, and a **real-time trading dashboard** with a modern, responsive UI.

## ✨ Features

- 🔐 **User Authentication** (Login & Signup)
- 💰 **Stock Buy & Sell** with quantity, price, and mode selection
- 📊 **Dashboard** to manage:
  - Orders
  - Positions
  - Profit & Loss (P&L)
- ⚡ Clean, responsive UI,bootstrap
- 🌍 Deployed across Vercel, Render, and Netlify
